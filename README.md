A lightweight client-side web app for compressing images directly in your browser.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- DaisyUI
- File-Saver
- JSZip
- Zustand

## Features
- Select images
- Adjust quality
- Set optional max width & height
- Instant client-side compression
